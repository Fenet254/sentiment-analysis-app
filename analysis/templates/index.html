How to Run Your Python File

Method 1: Using the Run Button

1. Open your Python file in VS Code

2. Click the "Run Python File" button (triangle icon) in the top-right corner

3. View output in the terminal panel

Method 2: Using Terminal

1. Open terminal in VS Code (Ctrl + `)

2. Navigate to your project directory

3. Run: python your_filename.py

Method 3: Using Python Command

1. Open terminal

2. Run: python -c "import your_filename"

Fixing the Import Error

The error occurs because CountVector doesn't exist in scikit-learn. You need to use CountVectorizer instead.

# Incorrect:
from sklearn.feature_extraction.text import CountVector
# Correct:
from sklearn.feature_extraction.text import CountVectorizer

After fixing the import, your code should run without errors.
